# ✦ LinkPage

A self-hostable personal link-in-bio page builder. Customize your profile, add unlimited links, track clicks, and switch between 5 animated themes.

## ✨ Features
- **Profile editor** — Set your name, bio, and avatar emoji
- **Unlimited links** — Add, edit, and delete links with icon, title, URL, and description
- **Click tracking** — Each link shows how many times it's been clicked
- **Visit counter** — Tracks total profile views across sessions
- **5 themes** — Purple, Ocean, Sunset, Forest, Rose — each with animated gradient + orbs
- **Animated background** — Floating color orbs that slowly drift
- **One-click share** — Copy your page URL to clipboard
- **Drag-friendly layout** — Clean card layout with hover animations
- **100% local** — All data stored in localStorage, no backend needed

## 🚀 Live Demo
Enable GitHub Pages: **Settings → Pages → Branch: master → Save**

## 🛠 Tech Stack
- Pure HTML, CSS, JavaScript
- Google Fonts: Plus Jakarta Sans
- Glassmorphism card design
- CSS animated gradient orbs

## 🎯 How to Use
1. Click **✏️ Edit Profile** to set your name, bio, and emoji
2. Click the **+ Add Link** button to add your first link
3. Fill in icon (emoji), title, URL, and optional description
4. Switch themes using the colored dots under your bio
5. Click **🔗 Share** to copy your page link
6. Enable **GitHub Pages** in repo settings to go live!
